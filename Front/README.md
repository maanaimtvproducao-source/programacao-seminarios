# Site Programação de Seminários (HTML/CSS/JS)

## Como usar
1. Extraia o zip.
2. Abra `index.html` no navegador.

> Dica: se o navegador bloquear algumas funcionalidades (clipboard), rode um servidor local:
- Python: `python -m http.server 5173`
- Acesse: http://localhost:5173
